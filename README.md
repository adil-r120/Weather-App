## How to Use
1. Enter the name of a city in the search box
2. Click the search button or press Enter
3. View the current weather information for the selected city
4. The app will display:
   - Current temperature
   - Weather description
   - Feels-like temperature
   - Humidity percentage
   - Wind speed

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/adil-r120/Weather-App.git
   ```
2. Obtain an API key from [OpenWeatherMap](https://openweathermap.org/api)
3. Replace the placeholder API key in `weather.js` with your own key
4. Open `weather.html` in your browser

## API Key Configuration
To use this app, you need to obtain a free API key from OpenWeatherMap:
1. Visit [OpenWeatherMap API](https://openweathermap.org/api)
2. Sign up for a free account
3. Copy your API key
4. In `weather.js`, find the line:
   ```javascript
   const apiKey = "YOUR_API_KEY_HERE";
   ```
5. Replace `YOUR_API_KEY_HERE` with your actual API key
